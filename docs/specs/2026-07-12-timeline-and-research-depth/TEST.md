# 测试方案

## 1. 事件脉络

- 事件按最近进展时间倒序并正确分到年月；
- 页面包含年、月和月内事件数；
- 搜索/主线/论文筛选会隐藏空月份；
- 桌面点击卡片打开右侧抽屉，移动端打开底部抽屉；
- Esc、背景、关闭按钮可关闭，焦点回到卡片；
- `?event=` 可直接打开指定事件。

## 2. GitHub Star

- 新鲜环境变量不发网络请求；
- 缺失变量时非测试构建读取官方 API 数量；
- 超时、非 200 或非法 payload 诚实降级；
- HTML 使用 GitHub 图标、Star 标签和独立数字段。

## 3. 研究门禁

- 高质量 Agent/LLM benchmark 论文超过事件候选阈值；
- 与核心主线无关或摘要过薄的论文保持 deferred；
- 新论文事件只进入 review，不自动发布占位内容；
- research Event 技术分析过薄时 readiness blocked；
- 六个 curated 研究事件均有一手 arXiv 链接、预印本提示、技术结果与下一验证。

## 4. 回归

- `npm run check`；
- 121+ 静态 HTML 内链检查；
- 静态隐私扫描；
- 1440×900、390×844 浏览器 smoke；
- Timeline 抽屉和 Star 组件无控制台错误。

## 5. 论文可见性与聚合

- 所有公开 `research` / `paper` Event 都能被“论文与研究”筛选命中，不因未达到首页精选深度而隐藏；
- 160 字符以上、具备明确贡献与核心领域相关性的 Tier 1 论文可进入 review；更薄或不相关内容继续 deferred；
- 同一 UTC 日期 4 篇以上论文渲染为论文日报，展开后 Event 数量与 slug 不丢失；
- 选择论文筛选或搜索命中时聚合组自动展开，空组和空月份继续隐藏；
- 首页研究精选仍只消费达到高质量分析深度的论文；
- 1440px 下事件脉络标题为单行，390px 下无横向滚动。

## 6. 全站证据抽屉与空月份回归

- 研究筛选下，无匹配卡片的 2026 年 5 月月份和对应空年份不可见；有匹配卡片的 2026 年 6 月显示卡片内容；
- 首页、趋势页、研究精选、Timeline、相关事件和 Scout 证据入口均能在当前页面打开抽屉；
- 普通点击不跳转，完整事件按钮和 Cmd/Ctrl 点击保留永久链接导航；
- 首次打开只请求一次 Timeline DTO，连续切换事件不重复请求；
- 初始 HTML 不再包含 50 份隐藏 `data-event-panel`，Timeline HTML 体积明显下降；
- 加载、失败和不存在状态均可见，Esc、遮罩、关闭按钮、URL 与焦点恢复继续工作；
- 抽屉桌面和移动端无横向溢出，无控制台错误，文本通过安全 DOM API 渲染。
